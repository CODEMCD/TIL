# 13. Roman to Integer
- [문제 링크](https://leetcode.com/problems/roman-to-integer/)

## 문제 풀이
- 단순히 if-else로 풀기

```cpp
class Solution {
public:
    int romanToInt(string s) {
        int ans = 0;
        for (int i=0; i<s.size(); ++i) {
            if (s[i] == 'I') {
                if ((i+1) < s.size() && s[i+1] == 'V') {
                    ans += 4;
                    ++i;
                }
                else if ((i+1) < s.size() && s[i+1] == 'X') {
                    ans += 9;
                    ++i;
                }
                else {
                    ans += 1;
                }
            }
            else if (s[i] == 'X') {
                if ((i+1) < s.size() && s[i+1] == 'L') {
                    ans += 40;
                    ++i;
                }
                else if ((i+1) < s.size() && s[i+1] == 'C') {
                    ans += 90;
                    ++i;
                }
                else {
                    ans += 10;
                }
            }
            else if (s[i] == 'C') {
                if ((i+1) < s.size() && s[i+1] == 'D') {
                    ans += 400;
                    ++i;
                }
                else if ((i+1) < s.size() && s[i+1] == 'M') {
                    ans += 900;
                    ++i;
                }
                else {
                    ans += 100;
                }
            }
            else if (s[i] == 'V') {
                ans += 5;
            }
            else if (s[i] == 'L') {
                ans += 50;
            }
            else if (s[i] == 'D') {
                ans += 500;
            }
            else if (s[i] == 'M') {
                ans += 1000;
            }
        }
        
        return ans;
    }
};
```

- if-else 최소로 사용하기
    - 로마 숫자의 규칙성을 찾으면 현재 로마 문자보다 한 칸 뒤의 로마 문자의 숫자 크기가 크면 현재 로마 문자는 뒤의 로마 문자까지 합쳐진 'IV'와 같은 문자라는 것을 알 수 있다. 이는 (뒤의 로마 문자 - 현재 로마 문자)로 볼 수 있다.

```cpp
class Solution {
public:
    int romanToInt(string s) {
        unordered_map<char, int> ri = {{'I', 1},
                                       {'V', 5},
                                       {'X', 10},
                                       {'L', 50},
                                       {'C', 100},
                                       {'D', 500},
                                       {'M', 1000}};
        
        int ans = 0;
        for (int i=0; i<s.size()-1; ++i) {
            if (ri[s[i]] < ri[s[i+1]]) {
                ans -= ri[s[i]];
            }
            else {
                ans += ri[s[i]];
            }
        }
        
        ans += ri[s[s.size()-1]];
        
        return ans;
    }
};
```
