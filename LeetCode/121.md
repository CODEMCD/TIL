# Best Time to Buy and Sell Stock
- [문제 링크](https://leetcode.com/problems/best-time-to-buy-and-sell-stock/)

## 문제 풀이
- 완전탐색

```c++
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int size = prices.size();
        
        int ans = 0;
        for (int i=0; i<size; ++i) {
            for (int j=0; j<size; ++j) {
                if (i > j) continue;
                else {
                    int cur = prices[j] - prices[i];
                    ans = ans > cur ? ans : cur;
                }
            }
        }
        
        return ans;
    }
};
```

```
Memory Usage: 9.6 MB
```

- DP

```c++
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int size = prices.size();
        int min = INT_MAX;
        int max = 0;
        for (int i=0; i<size; ++i) {
            if (prices[i] < min)
                min = prices[i];
            else if (prices[i] - min > max)
                max = prices[i] - min;
        }
        
        return max;
    }
};
```

```
Runtime: 8 ms
Memory Usage: 9.6 MB
```
